Creates and manages transactions

At minimum, an amount, credit card number, and credit card expiration date are required.

  1.  Braintree_Transaction::saleNoValidate(array(

  4.        'number' => '5105105105105100',
  5.        'expirationDate' => '05/12',
  6.        ),
  7.    ));

== Storing in the Vault ==

The customer and credit card information used for a transaction can be stored in the vault by setting transaction[options][storeInVault] to true.

  1.    $transaction = Braintree_Transaction::saleNoValidate(array(
  2.      'customer' => array(
  3.        'firstName'    => 'Adam',
  4.        'lastName'    => 'Williams'
  5.      ),
  6.      'creditCard'    => array(
  7.        'number'    => '5105105105105100',
  8.        'expirationDate'    => '05/2012'
  9.      ),
  10.      'options'    => array(
  11.        'storeInVault'    => true
  12.      )
  13.    ));
  14.  
  15.   echo $transaction->customerDetails->id
  16.   // '865534'
  17.   echo $transaction->creditCardDetails->token
  18.   // '6b6m'

To also store the billing address in the vault, pass the addBillingAddressToPaymentMethod option.

  1.    Braintree_Transaction.saleNoValidate(array(
  2.     ...
  3.      'options' => array(
  4.        'storeInVault' => true
  5.        'addBillingAddressToPaymentMethod' => true
  6.      )
  7.    ));

== Submitting for Settlement==

This can only be done when the transction's status is authorized. If amount is not specified, the full authorized amount will be settled. If you would like to settle less than the full authorized amount, pass the desired amount. You cannot settle more than the authorized amount.

A transaction can be submitted for settlement when created by setting $transaction[options][submitForSettlement] to true.

  1.    $transaction = Braintree_Transaction::saleNoValidate(array(
  2.      'amount'    => '100.00',
  3.      'creditCard'    => array(
  4.        'number'    => '5105105105105100',
  5.        'expirationDate'    => '05/2012'
  6.      ),
  7.      'options'    => array(
  8.        'submitForSettlement'    => true
  9.      )
  10.    ));
