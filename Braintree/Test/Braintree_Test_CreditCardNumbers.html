Class Braintree_Test_CreditCardNumbers

Description
Description | Vars (details) | Methods (details)

Credit card information used for testing purposes

The constants contained in the Braintree_Test_CreditCardNumbers class provide credit card numbers that should be used when working in the sandbox environment. The sandbox will not accept any credit card numbers other than the ones listed below.
