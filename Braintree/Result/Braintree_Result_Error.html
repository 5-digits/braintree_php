Braintree Error Result

An Error Result will be returned from gateway methods when the gateway responds with an error. It will provide access to the original request. For example, when voiding a transaction, Error Result will respond to the void request if it failed:

  1.  <?php
  2.  $result = Braintree_Transaction::void('abc123');
  3.  if ($result->success) {
  4.      // Successful Result
  5.  } else {
  6.      // Braintree_Result_Error
  7.  }
  8.  ?>

  • property-read: array $params: original passed params
  • property-read: object $errors: Braintree_Error_ErrorCollection
  • property-read: object $creditCardVerification: credit card verification data
