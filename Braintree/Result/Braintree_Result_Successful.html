Braintree Successful Result

A Successful Result will be returned from gateway methods when validations pass. It will provide access to the created resource.

For example, when creating a customer, Braintree_Result_Successful will respond to customer like so:

  1.  <?php
  2.  $result = Braintree_Customer::create(array('first_name' => "John"));
  3.  if ($result->success) {
  4.      // Braintree_Result_Successful
  5.      echo "Created customer {$result->customer->id}";
  6.  } else {
  7.      // Braintree_Result_Error
  8.  }
  9.  ?>
