Class Braintree_PagedCollection

Description
Description | Methods (details)

PagedCollection is a container object for paged result data

stores and retrieves search results and aggregate data

example:

  1.  $result = Braintree_Customer::all();
  2.  
  3.  // retrieve page 2 of search results
  4.  $result->nextPage();
  5.  
  6.  echo $result->currentPageNumber();
  7.  // prints 2
  8.  
  9.  // get an item
  10.  $item = $result->firstItem();
  11.  
  12.  echo $item['id'];
